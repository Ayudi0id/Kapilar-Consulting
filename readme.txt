# Kapilar Website Project

## Cara Import Database
1. Buka phpMyAdmin
2. Buat database baru dengan nama: kapilar_db
3. Klik Import
4. Pilih file: database/kapilar_db.sql
5. Klik Go

## Cek koneksi 
Pindahkan folder project ke C:\xampp\htdocs\ProjectKapilar(atau tempat install xampp kamu jika beda)

contohnya begini

C:\xampp\htdocs\ProjectKapilar\
    index.html
    contact.html
    cek-pt.html
    process_form.php
    process_pt.php
    test_db.php
    css/
    js/
    images/


## Nyalakan XAMPP

Apache → ON

MySQL → ON

## Test koneksi database

Buka browser lalu akses: http://localhost/ProjectKapilar/test_db.php
